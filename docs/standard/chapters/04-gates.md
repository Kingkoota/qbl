# Chapter 4: Gate Algebra

## 4.1 Mathematical Foundation

All QBL gates are elements of U(d^n) — the unitary group on n qudits of dimension d. The gate algebra is organized in layers:

```
Layer 0: Identity (trivial)
Layer 1: Generalized Pauli Group (d² generators per qudit)
Layer 2: Clifford Group (maps Paulis → Paulis)
Layer 3: Universal Set (Clifford + T gate)
Layer 4: Arbitrary Unitaries (synthesis from universal set)
```

## 4.2 Layer 1: Generalized Pauli Group

For dimension d with ω = e^(2πi/d):

### 4.2.1 Shift Operator (X)

```
SHIFT(q, k)  ≡  X^k|j⟩ = |j+k mod d⟩
```

| Dimension | Matrix |
|-----------|--------|
| d=2 | Standard Pauli-X (bit flip) |
| d=13 | 13×13 cyclic permutation matrix |

### 4.2.2 Clock Operator (Z)

```
CLOCK(q, k)  ≡  Z^k|j⟩ = ω^(jk)|j⟩
```

| Dimension | Matrix |
|-----------|--------|
| d=2 | Standard Pauli-Z (phase flip) |
| d=13 | diag(1, ω, ω², ..., ω¹²) |

### 4.2.3 Weyl Displacement

```
WEYL(q, a, b)  ≡  D(a,b) = τ^(ab) X^a Z^b
```

where τ = ω^((d+1)/2) for odd prime d.

**Key property:** For prime d, the d² operators {D(a,b) : a,b ∈ Z_d} form a complete orthogonal basis for all d×d matrices:

```
Tr(D(a,b)† D(a',b')) = d · δ_{aa'} · δ_{bb'}
```

### 4.2.4 Commutation Relations

```
X^a Z^b = ω^(-ab) Z^b X^a
D(a,b) D(c,d) = ω^(ad-bc) D(c,d) D(a,b)
```

## 4.3 Layer 2: Clifford Group

### 4.3.1 Quantum Fourier Transform (QFT)

```
QFT(q)  ≡  F|j⟩ = (1/√d) Σ_k ω^(jk)|k⟩
```

Conjugation action: `F X F† = Z`,  `F Z F† = X†`

For d=2: QFT = Hadamard gate H.

### 4.3.2 Phase Gate

```
PHASE(q, k)  ≡  S^k|j⟩ = ω^(kj(j-1)/2)|j⟩
```

Conjugation action: `S X S† = ω·XZ`

### 4.3.3 SUM Gate (Controlled-X / CNOT analog)

```
SUM(control, target)  ≡  |j⟩|k⟩ → |j⟩|j+k mod d⟩
```

For d=2: SUM = CNOT.

### 4.3.4 Controlled-Phase (CZ analog)

```
CPHASE(q0, q1)  ≡  |j⟩|k⟩ → ω^(jk)|j⟩|k⟩
```

### 4.3.5 SWAP

```
DSWAP(q0, q1)  ≡  |j⟩|k⟩ → |k⟩|j⟩
```

### 4.3.6 Modular Multiplication

```
MULT(q, a)  ≡  |j⟩ → |aj mod d⟩     (requires gcd(a,d)=1)
```

For d=13 (prime): all a ∈ {1,2,...,12} are valid multipliers.

### 4.3.7 Clifford Group Generation

**Theorem:** For prime d, the Clifford group C_n(d) on n qudits is generated by:
- QFT (single-qudit)
- PHASE (single-qudit)
- SUM (two-qudit)

All operations in Section 4.3 are derivable from these three generators.

## 4.4 Layer 3: Universal Gate Set

### 4.4.1 T Gate (Cubic Phase)

```
T(q)  ≡  T|j⟩ = ω^(j³/3)|j⟩   (mod d arithmetic)
```

For d=2: standard T gate = diag(1, e^(iπ/4))
For d=13: T|j⟩ = ω^(9j³ mod 13)|j⟩  (where 9 = 3⁻¹ mod 13)

**Universality theorem:** {QFT, PHASE, SUM, T} generates a dense subset of U(d^n) for any prime d.

### 4.4.2 Rotation Gates

```
ROT(q, a, b, θ, φ)  ≡  rotation in |a⟩-|b⟩ subspace by angle θ with phase φ
```

Any single-qudit unitary decomposes into O(d²) such rotations.

## 4.5 Gate Composition

```qbl
// Sequential application (right to left)
gate {
    QFT(q[0])
    SHIFT(q[0], 3)
    CLOCK(q[0], 7)
}
// Equivalent to: CLOCK(7) · SHIFT(3) · QFT applied to q[0]

// Pipeline syntax (left to right, more readable)
q[0] |> QFT |> SHIFT(3) |> CLOCK(7)

// Adjoint (reverse + conjugate transpose)
q[0] <| QFT <| SHIFT(3)
// Equivalent to: SHIFT(3)† · QFT†
```

## 4.6 Gate Properties Table

| Gate | Dimension | Parameters | Clifford? | Inverse |
|------|-----------|------------|-----------|---------|
| SHIFT(k) | any d | k ∈ Z_d | ✓ | SHIFT(d-k) |
| CLOCK(k) | any d | k ∈ Z_d | ✓ | CLOCK(d-k) |
| QFT | any d | — | ✓ | QFT† |
| PHASE(k) | any d | k ∈ Z_d | ✓ | PHASE(d-k) |
| SUM | any d | — | ✓ | SUM† |
| CPHASE | any d | — | ✓ | CPHASE† |
| MULT(a) | prime d | gcd(a,d)=1 | ✓ | MULT(a⁻¹) |
| ROT(a,b,θ,φ) | any d | a,b,θ,φ | ✗ (generic) | ROT(a,b,-θ,-φ) |
| T | prime d | — | ✗ | T† |
| H | d=2 only | — | ✓ | H (self-inverse) |
| CNOT | d=2 only | — | ✓ | CNOT |
