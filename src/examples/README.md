# QBL Examples

Collection of QBL programs demonstrating language features.

## Basic
- `bell.qbl` — Bell state (entanglement)
- `teleport.qbl` — Quantum teleportation protocol
- `ghz.qbl` — GHZ state (multi-qubit entanglement)
- `grover.qbl` — Grover's search algorithm (2 qubits)

## Advanced (v0.2.0)
- `protected.qbl` — Autonomous error correction
- `feedback.qbl` — Latency-bounded classical feedback
- `multi_backend.qbl` — Hardware-agnostic pulse control
